# RecyclerViewDatabindingAdapter

## Step 1.JitPack库添加到你的构建文件
  <pre>allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}  </pre>
  
  ## Step 2. app gradle 添加依赖
  <pre>
  dependencies {
	        compile 'com.github.zhangshuqi:RecyclerViewDatabindingAdapter:v1.0.2'
          }
     </pre>
